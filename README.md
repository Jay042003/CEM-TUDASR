# Project Usage Guide

This repository provides scripts for **training** and **testing** your model. Follow the instructions below to get started.

---

---

##  Training the Model

To train the model, run:

```bash 
./training.sh 
```
to test the model, run:

```bash 
./testing.sh 
```
to test the model downsampling model, run:

```bash 
./testing_down.sh 
```

## Notes
Make sure all scripts have execution permission by running

```bash 
chmod +x training.sh testing.sh testing_down.sh
```
